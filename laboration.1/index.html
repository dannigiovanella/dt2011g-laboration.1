Laboration 1

Automatisering & publicering

Detta är webbplats skapad för laboration 1 i kursen Frontend-baserad webbutveckling - DT2011G

Syftet med uppgiften är att visa på hur man kan underlätta i utvecklingsarbetet genom att använda verktyg som automatiserar optimering av bland annat kod-filer, för att minimera kod och för bilder för att få ner bildstorleken.

Utöver automatiseringen av optimering så visar denna uppgift på hur man kan jobba med verktyg för att effektivisera publiceringsprocessen där änrdingar på webblatsen automatiskt uppdateras på den publicerade sidan.

Denna webbplats består av tre sidor.

  • Startsida - Presentation av uppgift
  • Bilder - Presentation av fem olika optimerade bilder
  • Process - En beskrivning av arbetsprocessen

Webbplatsen är under utvecklingsarbetet versionshanterad med Git kopplad mot Github för dokumentation och publicering.
